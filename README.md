# Proyecto 1 - Predicción Titanic
Mi primer proyecto de Data Science. Como objetivo, predecir la supervivencia de los pasajeros del Titanic usando un modelo de Machine Learning.
Este proyecto ha sido desarrollado en Kaggle. Es uno de los retos que se plantean al iniciarse en ML.

### Proceso del proyecto
* **Exploración:** Análisis inicial de las variables dadas
* **Limpieza de datos:** Gestión de valores nulos y borrado de columnas que aportaban poca info
* **Feature Engineering:** Conversión de variables categóricas
* **Modelado:** Entrenamiento de un modelo base 'DecisionTreeClassifier'
* **Predicción:** Limpieza de los datos del 'test' y generación de las predicciones

### Resultados
* **Precisión (Entrenamiento):** 98.20% (No es un resultado realista -> explicado en el Notebook)
* **Puntuación Real (Kaggle):** 73.68%
